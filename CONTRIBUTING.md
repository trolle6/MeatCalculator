# Contributing

Thanks for your interest in Smoke Lab.

## Bugs and ideas

Open an [issue](https://github.com/trolle6/SmokeLab/issues) with:

- What you expected vs what happened
- Browser and device (if UI-related)
- A share URL from **Copy link** if it’s about saved cook settings

## Pull requests

Small, focused PRs are welcome. Please run locally before submitting:

```bash
cd "meat calculator"
dotnet build
node --check wwwroot/app.js
```
